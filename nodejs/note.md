#Node.js quick file server (static files over HTTP)

A good "ready-to-use tool" option could be http-server:

npm install http-server -g
To use it:

cd D:\Folder
http-server
Or, like this:

http-server D:\Folder
Check it out: https://github.com/nodeapps/http-server
